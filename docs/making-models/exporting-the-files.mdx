Once you have created your model, you'll need to export it and place the exported files in the correct location for GeckoLib to see them.

## Exporting the JSON files

To export, click the below menu paths:<br/>
Model: `File` -> `Export` -> `Export GeckoLib Model`<br/>
Animations: `File` -> `Export` -> `Export GeckoLib Animations`<br/>
Display Settings: `File` -> `Export` -> `Export GeckoLib Display Settings`

![Blockbench Export](/img/blockbench_export.png)

## Exporting the texture

To export the texture, you'll need to right click on the texture and click `Save As`

![Blockbench Export](/img/blockbench_texture_export.png)
