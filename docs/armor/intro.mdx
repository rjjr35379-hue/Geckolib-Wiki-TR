GeckoLib supports rendering and animating custom armor models.

To do so, you will need to create a standard item, then add the GeckoLib armor rendering parts onto it.

In total, you will need to complete the following steps:
1. Create the item class
2. Register the item
3. Create the armor renderer class
4. Apply the armor renderer

When you're ready to begin, click `Next` at the bottom of the page to continue.
