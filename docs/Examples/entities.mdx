GeckoLib has eight example entities for references.

### Bat
The Bat is an example entity that showcases basic conditional animation controllers, and spawning particles at a bone position.

- [Entity Class](https://github.com/bernie-g/geckolib-examples/blob/Multiloader-1.21.11/common/src/main/java/com/example/examplemod/entity/BatEntity.java)
- [GeoModel Class](https://github.com/bernie-g/geckolib-examples/blob/Multiloader-1.21.11/common/src/main/java/com/example/examplemod/client/model/entity/BatModel.java)
- [Renderer Class](https://github.com/bernie-g/geckolib-examples/blob/Multiloader-1.21.11/common/src/main/java/com/example/examplemod/client/renderer/entity/BatRenderer.java)

### Vehicle
The Vehicle is an example entity that showcases some more specific molang queries in animations.

- [Entity Class](https://github.com/bernie-g/geckolib-examples/blob/Multiloader-1.21.11/common/src/main/java/com/example/examplemod/entity/BikeEntity.java)
- [GeoModel Class](https://github.com/bernie-g/geckolib-examples/blob/Multiloader-1.21.11/common/src/main/java/com/example/examplemod/client/model/entity/BikeModel.java)
- [Renderer Class](https://github.com/bernie-g/geckolib-examples/blob/Multiloader-1.21.11/common/src/main/java/com/example/examplemod/client/renderer/entity/BikeRenderer.java)

### GeoRenderLayers Entity
The GeoRenderLayers Entity is an example entity that showcases using GeoRenderLayers to render another layer over the model,
as well as using an inline `DefaultedEntityGeoModel`.

- [Entity Class](https://github.com/bernie-g/geckolib-examples/blob/Multiloader-1.21.11/common/src/main/java/com/example/examplemod/entity/CoolKidEntity.java)
- [Renderer Class](https://github.com/bernie-g/geckolib-examples/blob/Multiloader-1.21.11/common/src/main/java/com/example/examplemod/client/renderer/entity/CoolKidRenderer.java)

### Mutant Zombie Entity
The Mutant Zombie is an example entity that showcases held-item rendering and custom bone texture rendering

- [Entity Class](https://github.com/bernie-g/geckolib-examples/blob/Multiloader-1.21.11/common/src/main/java/com/example/examplemod/entity/DynamicExampleEntity.java)
- [GeoModel Class](https://github.com/bernie-g/geckolib-examples/blob/Multiloader-1.21.11/common/src/main/java/com/example/examplemod/client/model/entity/MutantZombieModel.java)
- [Renderer Class](https://github.com/bernie-g/geckolib-examples/blob/Multiloader-1.21.11/common/src/main/java/com/example/examplemod/client/renderer/entity/MutantZombieRenderer.java)

The Fake Glass entity is an example entity that showcases dynamically using existing textures as bone textures.
