GeckoLib has two example armors for references.

### Gecko Armor
The Gecko Armor is a set of armor that showcases GeckoLib armor rendering, `DataTicket` usage, and glowmasks.

[Item Class](https://github.com/bernie-g/geckolib-examples/blob/Multiloader-1.21.11/common/src/main/java/com/example/examplemod/item/GeckoArmorItem.java) |
[Renderer Class](https://github.com/bernie-g/geckolib-examples/blob/Multiloader-1.21.11/common/src/main/java/com/example/examplemod/client/renderer/armor/GeckoArmorRenderer.java)

### Wolf Armor
The Wolf Armor is a set of armor that showcases GeckoLib armor rendering, and `DataTicket` usage.
