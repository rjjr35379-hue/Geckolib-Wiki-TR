Before getting started with setting up your development environment with GeckoLib, you'll need to identify which
version of GeckoLib you will be using.

## Finding the version
You'll need to visit either `Modrinth` or `CurseForge` to find the version you want to use.

<Tabs groupId="website" queryString="website">
    <TabItem value="curseforge" label="CurseForge">
        1. Go to the [GeckoLib CurseForge Page](https://www.curseforge.com/minecraft/mc-mods/geckolib)
        2. From there, click on the `Files` tab

        <img src={"/img/curseforge_mainpage.png"} alt="CurseForge Main Page" width={"600px"} />

        3. Filter for the Minecraft version you're using (`1.21.11`, `26.1`, etc.)

        <img src={"/img/curseforge_version_filter.png"} alt="CurseForge Version Filter" width={"600px"} />

        4. Filter for the mod loader you want (`Fabric`, `Forge`, `NeoForge`)

        <img src={"/img/curseforge_platform_filter.png"} alt="CurseForge Platform Filter" width={"600px"} />

        5. Note the top entry in the list, which shows the version number you want to use.

        <img src={"/img/curseforge_found_version.png"} alt="CurseForge Found Version" width={"800px"} />

        6. Note this version number down, you'll need it for the next step
    </TabItem>
    <TabItem value="modrinth" label="Modrinth">
        1. Go to the [GeckoLib Modrinth Page](https://modrinth.com/mod/geckolib)
        2. From there, click on the `Versions` tab

        <img src={"/img/modrinth_mainpage.png"} alt="Modrinth Main Page" width={"600px"} />

        3. Filter for the mod loader you want (`Fabric`, `Forge`, `NeoForge`)

        <img src={"/img/modrinth_platform_filter.png"} alt="Modrinth Platform Filter" width={"600px"} />

        4. Filter for the Minecraft version you're using (`1.21.11`, `26.1`, etc.)

        <img src={"/img/modrinth_version_filter.png"} alt="Modrinth Version Filter" width={"600px"} />

        5. Note the top entry in the list, which shows the version number you want to use.

        <img src={"/img/modrinth_found_version.png"} alt="Modrinth Found Version" width={"800px"} />

        6. Note this version number down, you'll need it for the next step
    </TabItem>
</Tabs>
