Bone Updaters are GeckoLib's interface for allowing bones to be manipulated when rendering.

## What are they?
Bone updaters are a new function that can be registered to a `RenderPassInfo` to allow for updating a bone's position, rotation, or scale for rendering.

This is different to how GeckoLib used to work, where you had direct access to the `GeoBone` to modify it.
Now, all modifications to the bone must be performed via a `Bone Updater`, so that the `RenderPassInfo` can bake the changes.
