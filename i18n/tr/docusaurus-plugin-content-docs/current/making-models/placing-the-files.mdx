GeckoLib'in dosyalarınızı bulabilmesi için, bu dosyaların doğru konuma yerleştirilmesi gerekir.

Dosyalar yerleştirildikten sonra, bunlara kod içinde başvurmak için bir `GeoModel` örneği kullanılır.<br/>
Daha fazla bilgi için [GeoModels](/concepts/geomodels/overview.mdx) Sayfasına bakın

## Dosya Konumları
### Model Dosyaları
GeckoLib5'te model dosyaları (`.geo.json`) `assets/<modid>/geckolib/models/` Dosyasına yerleştirilir.

![Model dosyalarını](/img/geckolib_models_folder.png)

Modellerinizi düzenlemek için alt klasörler kullanabilirsiniz; ayrıca bazı GeckoLib özellikleri (örneğin `DefaultedGeoModel` sınıfları) alt klasörleri otomatik olarak kullanacaktır.

:::tip
'.geo' uzantısı tamamen opsiyonel'dir 
:::

### Animation Files
GeckoLib5'te animasyon dosyaları (`.animation.json`) `assets/<modid>/geckolib/animations/` Dosyasına yerleştirilir.

![Animasyon Dosyaları](/img/geckolib_animations_folder.png)

Animasyonlarınızı düzenlemek için alt klasörler kullanabilirsiniz; ayrıca bazı GeckoLib özellikleri (örneğin `DefaultedGeoModel` sınıfları) alt klasörleri otomatik olarak kullanacaktır.

:::tip
'.animation' uzantısı tamamen opsiyonel'dir 
:::

### Texture Files
Geckolib5'te, texture dosyaları `assets/<modid>/textures/` Dosyasına konulması lazımdır, farkı bir texture'a yapcağın şekilde aynı yapılması lazımdır.

Texturelarınızı düzenlemek için alt klasörler kullanabilirsiniz; ayrıca bazı GeckoLib özellikleri (örneğin `DefaultedGeoModel` sınıfları) alt klasörleri otomatik olarak kullanacaktır.
