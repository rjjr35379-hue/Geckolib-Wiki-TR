Modelinizi oluşturduktan sonra, bunları dışa aktarmanız ve GeckoLib'in görebilmesi için dışa aktarılan dosyaları doğru konuma yerleştirmeniz gerekecektir.

## Json dosyalarını Dışa aktarma

Dışa aktarmak için, Aşağıdaki menü butonlarına basınız:<br/>
Model: `File` -> `Export` -> `Export GeckoLib Model`<br/>
Animasyon: `File` -> `Export` -> `Export GeckoLib Animations`<br/>
Görüntü Ayarları: `File` -> `Export` -> `Export GeckoLib Display Settings`

![Blockbench Export](/img/blockbench_export.png)

## texture'ı export etmek

Texture'ı dışa aktarmak için, texture'ınızın üzerine sağ mouse butonuyla basıp `Save As`'e tıklamanız lazımdır

![Blockbench Export](/img/blockbench_texture_export.png)
